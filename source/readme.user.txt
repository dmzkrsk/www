Как вернуть проект к жизни
=================================
1. Перейдите в папку с исходниками
2. Если в вашей поставке отсутствуют уже откомпилированые проекты из 
   папки DLLs  (это  можно понять  по  отсутствию  dll-ок и lib-ов в 
   папках  Debug и  Release),  то  выполните  следущие  шаги,  иначе
   перейдите к пункту 5
3. Запустите www_devel.dsw
4. Откомпилируйте  Debug  и  Release  версии  библиотек  в  следущем 
   порядке
     -b64
     -mime
     -cmdparser
     -fsys
     -linkex
     -curl(*)
     -urld(*)
   ПЕРЕД компиляцией проектов отмеченых (*) запустите update.bat
5. Запустите update.bat
6. Теперь проект полностью собран
==================================
update.bat
==================================
Этот файл служит для копирования новых фалов библиотек (.dll и .lib),
а также h-файлов в папки lib и librel, откуда  их  читает  компилятор
при сборке зависящих проектов. Также он  обнавляет dll-ки в  корневой
папке проекта для правильного запуска www.exe
Если  вы добавляете  свои dll или меняете структуру, то позабодтесь о 
соответствующих изменениях в update.bat
